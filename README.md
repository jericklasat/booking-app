<h2>Installation</h2>
<ul>
  <li>install and run docker</li>
  <li>create .env file based on .env.example</li>
  <li>sail build</li>
  <li>sail up -d</li>
  <li>sail php artisan migrate</li>
</ul>

<h2>To Re-Run the program</h2>
<ul>
  <li>Run: sail up -d</li>
  <li>If you want to view logs in the terminal run: sail up</li>
</ul>
